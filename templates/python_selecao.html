estruturas de seleção (if, elif, else)

o que é isso?

basicamente, estruturas de seleção permitem que seu programa tome decisões. ele verifica se uma condição é verdadeira e, dependendo do resultado, executa um bloco de código ou outro. em python, usamos if para testar a primeira condição, elif (que é uma abreviação de "else if") para testar condições adicionais se as anteriores forem falsas, e else para um bloco de código que roda se nenhuma das condições anteriores for verdadeira.

quando usar?

você usa isso o tempo todo. por exemplo, para verificar se um usuário digitou a senha correta, para determinar se um aluno passou de ano com base na nota, ou para mudar o comportamento de um jogo dependendo da ação do jogador. qualquer situação onde o programa precisa seguir caminhos diferentes baseado em alguma informação.

exemplo de código:


idade = 20
tem_carteira = True

if idade >= 18 and tem_carteira:
    print('pode dirigir.')
elif idade >= 18 and not tem_carteira:
    print('maior de idade, mas precisa tirar a carteira pra dirigir.')
else:
    print('não pode dirigir ainda.')

nota = 7.5
if nota >= 7.0:
    print('aprovado!')
elif nota >= 5.0:
    print('recuperação.')
else:
    print('reprovado.')
